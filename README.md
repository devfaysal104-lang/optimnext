# Optimnext Website

This is a single-page developer website for Optimnext.

## How to use with GitHub Pages

1. Create a new GitHub repository, for example: optimnext
2. Upload `index.html` to the repository.
3. Go to Settings → Pages.
4. Under "Build and deployment", choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.
6. Your website will be live at:
   https://YOUR-GITHUB-USERNAME.github.io/optimnext/

Use that URL in Google Play Console.
